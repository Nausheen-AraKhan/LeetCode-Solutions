This folder contains solutions to problems solved using the **Two Pointers** technique.

## What is the Two Pointers Technique?

Two Pointers is an algorithmic approach where two indices (or pointers) traverse a data structure, usually from opposite ends or in the same direction, to solve problems efficiently.

It is commonly used to:
- Reduce time complexity from **O(n²)** to **O(n)**
- Process sorted arrays efficiently
- Find pairs or triplets satisfying a condition
- Handle sliding windows and partitioning problems
