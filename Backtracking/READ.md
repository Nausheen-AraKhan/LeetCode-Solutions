Backtracking - LeetCode Solutions
📌 About
This repository contains my solutions to Backtracking problems from LeetCode. 
The goal is to strengthen recursion, decision-making, and state-space exploration through consistent practice.
Backtracking is a recursive algorithmic technique that explores all possible choices and undoes previous decisions whenever a path 
cannot lead to a valid solution.

🚀  PROBLEMS
* Problem 78: Subsets | MEDIUM


The Three Steps
* Choose a possible option.
* Explore recursively.
* Unchoose (Backtrack) to restore the previous state.

🎯 Learning Goals
* Understand recursive thinking.
* Master backtracking patterns.
* Improve problem-solving speed.
* Identify when to use backtracking over brute force.
* Build intuition for pruning unnecessary recursive calls.
